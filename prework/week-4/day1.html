2 Install Essential Extensions

Extensions add superpowers to VS Code. Let's install 5 must-have extensions for web development.

How to install extensions:

  1. Click the Extensions icon in the left sidebar (or press Ctrl/Cmd + Shift + X)
  2. Search for the extension name
  3. Click "Install"

Live Server

by Ritwick Dey

Opens HTML files in your browser with auto-reload when you save changes. Essential for web development!

Search: "Live Server Ritwick Dey"

Prettier

by Prettier

Automatically formats your code to make it clean and consistent. Saves tons of time!

Search: "Prettier - Code formatter"

ESLint

by Microsoft

Finds errors and potential problems in your JavaScript code before you run it.

Search: "ESLint"

Auto Rename Tag

by Jun Han

When you rename an HTML opening tag, it automatically renames the closing tag. Magic!

Search: "Auto Rename Tag"

Path Intellisense

by Christian Kohler

Autocompletes file paths as you type. Makes linking files much easier!

Search: "Path Intellisense"

3 Configure VS Code Settings

Let's customize VS Code for the best coding experience.

  1. Open Settings

    Press Ctrl + , (Windows/Linux) or Cmd + , (Mac)

  2. Open Settings JSON

    Click the icon in the top right that looks like a file with an arrow

    Or press Ctrl/Cmd + Shift + P and type "Open Settings (JSON)"

  3. Paste These Settings 
    {
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.wordWrap": "on",
  "editor.formatOnSave": true,
  "files.autoSave": "afterDelay",
  "editor.minimap.enabled": false,
  "workbench.colorTheme": "Default Dark+",
  "editor.fontFamily": "Consolas, 'Courier New', monospace"
}
  4. Save Settings

    Press Ctrl/Cmd + S to save

What do these settings do?

  • fontSize: Makes text easier to read
  • tabSize: Uses 2 spaces for indentation (industry standard)
  • wordWrap: Long lines wrap instead of scrolling horizontally
  • formatOnSave: Automatically formats code when you save
  • autoSave: Saves your work automatically after typing
  • minimap: Removes the mini code preview (cleaner interface)
